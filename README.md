# niri dotfiles

My [Niri](https://github.com/YaLTeR/niri) Wayland compositor configuration.

## Structure
- `config.kdl` — main Niri config
- `dms/` — modular config (keybinds, colors, layout, outputs, etc.)

## Usage
Clone into `~/.config/niri`:

```sh
git clone https://github.com/jhnhnsn/niri-config.git ~/.config/niri
```

## Per-machine notes
- `dms/outputs.kdl` contains monitor/display layout and will differ between machines — adjust after cloning.
